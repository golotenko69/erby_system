Добро пожаловать в Единую региональную базу учета (ЕРБУ)

ЕРБУ — это единая цифровая платформа для Забайкальского края, предназначенная для ведения централизованного учета, мониторинга обучения и содействия трудоустройству студентов с ограниченными возможностями здоровья (ОВЗ) и инвалидностью.

🚀 Быстрый старт: Как работать с системой?

👨‍🎓

Вкладка «Студенты»

Здесь вы можете добавлять новых студентов, редактировать личные дела, вносить данные паспорта, инвалидности (МСЭ, ПМПК), информацию об учебе и сведения о родителях.

📊

Конструктор отчетов

Нажмите кнопку «Сформировать отчет (Excel)» в журнале студентов, выберите галочками нужные для выгрузки поля, и система автоматически создаст готовый аналитический файл.

💼

Мониторинг занятости

В карточке каждого студента предусмотрен специальный блок для контроля его трудоустройства после выпуска (или указания официальной причины, если он не трудоустроен).
